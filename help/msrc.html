MSRC

Mapeador de Sistemas em Referências Cruzadas

Faz uma análise dos programas fontes COBOL. Os comandos CALL e COPY e as declarações de arquivo SELECT serão usadas como base para identificar todos os arquivos, books, programas e rotinas resultando em 6 relatórios de referência cruzada.

Formato: MSRC Programas.

Onde Programas é a especificação de um ou mais programas podendo conter caracteres "coringas". O comando deve ser executado no mesmo diretório onde se encontram todos os programas fontes:

Exemplo, usando como base o Sistema de Contabilidade gratuito Contador:

COBOLware C:\contador\SLB> MSRC CB*.CBL
