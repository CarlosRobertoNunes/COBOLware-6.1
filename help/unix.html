System

Informa a plataforma de execução.

Sintaxe

System OS-CODE;código-do-sistema

Argumentos

código-do-sistema

Variável de um byte que recebe o código que identifica o sistema operacional e a modalidade de execução..

Código

Ambiente

1

Unix, Linux, Aix ou HP/UX

2

MS/PC-DOS, PC-MOS, DR-DOS ou compatíveis

3

Microsoft Windows em tela DOS (16-bit)

4

Microsoft Windows em janela texto (16-bit)

5

Microsoft Windows em janela texto (32-bit)

6

Microsoft Windows em janela gráfica (16-bit)

7

Microsoft Windows em janela gráfica (32-bit

Exemplo

Codificação COBOL

77 sistema PIC X(001) VALUE SPACE.
88 GRAFICO VALUE "7"

EXEC COBOLware System
         OS-CODE;sistema
END-EXEC
IF GRAFICO
   PREFORM PROCESSOS-GRAFICOS
ELSE
   PERFORM PROCESSOS-TEXTO
END-IF
