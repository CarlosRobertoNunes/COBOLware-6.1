GetSystem

Obtém informações do sistema aplicativo

Sintaxe

GetSystem
[ AUTHENTICATION;flag-autenticação ]
[ DELETE;exclusão ]
[ DEVELOPER;licenciado ]
[ E-MAIL;e-mail-do-usuário-corrente ]
[ GROUP;grupo ]
[ INSERT;inclusão ]
[ LEVEL;nível ]
[ MODULE;módulo ]
[ PAGE;página ]
[ PASSWORD;senha ]
[ PROGRAM;programa ]
[ QUERY;consulta ]
[ REPORT-APLICATION;nome-do-sistema-em-relatórios ]
[ REPORT-COMPANY;nome-da-empresa-em-relatórios ]
[ SCREEN-APLICATION;nome-do-sistema-em-tela ]
[ SCREEN-COMPANY;nome-da-empresa-em-tela ]
[ SMTP;sevidor-de-envio-de-e-mail ]
[ SMTP-PORTporta-do-sevidor-de-envio-de-e-mail ]
[ SMTP-PASSWORD;senha-de-acesso-ao-smtp ]
[ SSL;flag-ssl ]
[ TASK;número-do-processo ]
[ UPDATE;alteração ]
[ USER;usuário-corrente ]

Argumentos

flag-autenticação

Indica com “1” se o servidor de envio de e-mail do usuário requer autenticação.

exclusão

Variável de 1 byte que recebe * indicando que o usuário corrente não tem direito de excluir com o programa corrente.

licenciado

Nome do licenciado do COBOLware (até 40 bytes).

e-mail-do-usuário-corrente

Endereço de e-mail configurado nos atributos do usuário corrente (até 42 bytes).

grupo

Nome do grupo do usuário corrente (até 22 bytes).  

inclusão

Variável de 1 byte que recebe * indicando que o usuário corrente não tem direito de incluir com o programa corrente.

nível

Número de nível de privilégio do usuário corrente (1 byte).

módulo

Nome do módulo (grupo de programas declarado na página do menu) da opção corrente (até 25 bytes).

página

Número da página de menu onde está configurada a opção corrente (até 6 bytes).

senha

Senha do usuário corrente (até 30 bytes).

programa

Nome do programa chamado pelo gerenciador na opção corrente (até 8 bytes).

consulta

Variável de 1 byte que recebe * indicando que o usuário corrente não tem direito a fazer consultas com o programa corrente.

nome-do-sistema-em-relatórios

Descrição do sistema configurada no gerenciador de sistemas para figurar nos cabeçlhos de relatórios gerados com o comando Print Report (até 30 bytes).

nome-da-empresa-em-relatórios

Identificação da empresa configurada no gerenciador de sistemas para figurar nos cabeçalhos de relatórios gerados com o comando Print Report (até 30 bytes).

nome-do-sistema-em-tela

Descrição do sistema configurada no gerenciador de sistemas para figurar em telas e no título da janela principal (até 30 bytes).

nome-da-empresa-em-tela

Identificação da empresa configurada no gerenciador de sistemas para figurar em telas e no título da janela principal (até 30 bytes).

sevidor-de-envio-de-e-mail

Servidor configurado para o envio de e-mails nos atributos do usuário corrente (até 42 bytes).

porta-do-servidor-de-envio-de-email

Informa o numero da porta utilizada utilizada no servidor de envio de e-mail.

senha-de-acesso-ao-smtp

Senha do e-mail do usuário encontrada nos atributos do usuário corrente (até 20 bytes).

flag-ssl

Indica se o servidor de envio de e-mail exige criptografia SSL.

número-do-processo

Número corrente do contador de execuções do gerenciador de sistemas (até 6 bytes).

alteração

Variável de 1 byte que recebe * indicando que o usuário corrente não tem direito a fazer alterações com o programa corrente.

usuário-corrente

Senha do usuário corrente.
