Pack String

Ajusta string para exibição.

Observação

O ajuste consiste em remover espaços e zeros não-significativos.

Sintaxe

Pack Sring mensagem
[ WIDTH tamanho ] 

Argumentos

mensagem

Campo de até 8192 caracteres contendo a string a ser ajustada.

tamanho

Campo numérico que recebe a largura da string ajustada.

Exemplo

Codificação COBOL

EXEC COBOLware Pack String mensagem
           WIDTH tamanho
 END-EXEC
