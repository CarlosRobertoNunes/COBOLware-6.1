Spool File

Transfere relatório de arquivo texto para .spool

Sintaxe

Spool FILE arquivo
[ REPORT  código-relatório ]
[ WIDTH largura-do-relatório]
[ TITLE título-do-relatório ]
[ NOTE comentário ]

Argumentos

arquivo

Nome do arquivo texto onde deve estar gravado o relatório a ser transferido para spool.

código-relatório

Identificação do relatório para efeito de configuração e spool.

largura-do-relatório

Largura máxima reconhecida para a linha de impressão, de 60 a 255 colunas, quando omitido é assumido largura 132.

título-do-relatório

Frase de até 34, 86 ou 174 caracteres de acordo com a largura de impressão para documentar o título do relatório no spool.

comentário

Mensagem de comentário na lista de relatórios no spool.

Exemplo

Codificação COBOL

EXEC COBOLware Spool FILE File-id-relatorio
           REPORT "TESTE"
           TITLE "Teste de Spool"
           NOTE "Teste do comando Spool File"
END-EXEC
