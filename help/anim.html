ANIM

Executor de depuração

Formatos:

1

ANIM Programa

Modo console API

2

ANIMG Programa

Modo gráfico

Onde Programa é o nome do programa a ser depurado.

Testar um programa sem precisar passar pelo Gerenciador de sistemas é um recurso interessante para facilitar o trabalho no ambiente de desenvolvimento

Os programas a serem depurados devem estar compilados com a opção Anim para que os fontes sejam "animados". Os programas compilados de outra forma serão executados normalmente, assim quando uma série de módulos já confiáveis precisam rodar antes de se atingir o módulo "suspeito" somente este deve estar compilado para depuração, isso faz um atalho até o problema.

Para evitar que se abra uma espécie de "porta dos fundos" que violaria a segurança do sistema no ambiente do usuário, será exigido usuário e senha. Este incômodo pode ser contornado com a variável de ambiente CWTEST=/U:Usuário /S:Senha /NOFRAME sendo que NOFRAME indica que a moldura do Gerenciador de sistemas não deve ser traçada no modo texto.
