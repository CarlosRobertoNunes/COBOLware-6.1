Teclas de edição

Visa melhor legibilidade de código no tratamento de retornos obtidos pelo comando GetKey. Supondo que o usuário tenha pressionado a tecla Esc, o item TECLA receberia o valor  027. Assim, para tratar deste evento, seria codificado   "IF TECLA = 027", utilizando COPY CWEDIT, a declaração passa para "IF EDIT-ESC"  facilitando a compreensão do código.
