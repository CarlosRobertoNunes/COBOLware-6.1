Print Report

Emissão de relatório.

Sintaxe

Print { REPORT  código-relatório | CLOSE }
[ WIDTH largura-do-relatório]
[ FREE | [ TITLE título-do-relatório ]
              [ SUBTITLE subtitulo-do-relatório ]
              [ HEADER(1...5) cabeçalho ] ]
[ DETAIL linha-de-detalhe ]
[ FULL decorrido EMPTY  previsto ]
[ NOTE comentário ]
[ CANCEL;indicador-de-processo-interrompido ]

Argumentos

código-relatório

Identificação do relatório para efeito de configuração e spool.

CLOSE

Encerra a impressão do relatório corrente.

largura-do-relatório

Largura máxima reconhecida para a linha de impressão, sendo suportado até 500 colunas, quando omitido é assumido largura 132.

FREE

Formato de impressão livre. O controle de salto de página e formatação de cabeçalhos serão controlados pelo programador na linha-de-detalh.

título-do-relatório

Frase de até 34, 86 ou 174 caracteres de acordo com a largura de impressão a ser impressa no topo de cada página.

subtitulo-do-relatório

Frase complementar a ser impressa na linha abaixo do título-do-relatório..

cabeçalho

Cabeçalho de identificação dos campos de detalhe (até 5).

linha-de-detalhe

Linha de detalhe do relatório ou na coluna 1 $Include arquivo-a-anexar (é apagado em seguida).

decorrido

Indica o progresso do processo de geração do relatório, pode ser usado quando a quantidade a processar é conhecida.

previsto

Indica a quantidade prevista no processamento

comentário

Mensagem de comentário na lista de relatórios no spool.

indicador-de-processo-interrompido

Variável de 1 byte que retorna o valor "9" se o usuário cancelar a emissão do relatório.
77 fim PIC X(001) VALUE SPACE.
88 END-PRINT VALUE "9".

Exemplo

Codificação COBOL

PERFORM VARYING I FROM 1 BY 1
                         UNTIL I > 100 OR END-PRINT
                   EXEC COBOLware Print Report "TESTE"
                              WIDTH 80
                              TITLE "Teste do comando PRINT"
                              SUBTITLE "Listando de 1 a 100"
                              HEADER(1) "I"
                              DETAIL I
                              FULL I EMPTY 100
                             CANCEL;fim
                   END-EXEC
END-PERFORM

IF   NOT END-PRINT
      EXEC COBOLware Print CLOSE END-EXEC
END-IF
