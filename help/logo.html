Logon/Login

Troca de usuário.

Sintaxe

Logon|Login
[ USER usuário ] 
[ PASSWORD senha ]
[ CANCEL;indicador-de-desistência ]

Argumentos

usuário

Nome do usuário a ser conectado, se omitido será solicitado em uma caixa de dialogo.

senha

Senha do usuário a ser conectado, se omitido será solicitado em uma caixa de dialogo.

indicador-de-desistência

Variável que retorna o valor "Y" se o usuário desistir de efetuar seu logon/login.
77 fim PIC X(001) VALUE SPACE.
88 CANCELED VALUE "Y".


Exemplo

Codificação COBOL

EXEC COBOLware Logon
           USER ”Fulano”
           PASSWORD ”pipoca”
END-EXEC
