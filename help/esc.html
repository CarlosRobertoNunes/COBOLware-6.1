Escape keys

Visa melhor legibilidade de código no tratamento de retornos obtidos pelo comando ACCEPT TECLA FROM ESCAPE KEY. Supondo que o usuário tenha pressionado a tecla Esc, o item TECLA receberia o valor  01. Assim, para tratar este evento, seria codificado   "IF TECLA = 01", utilizando CWKEYS, a declaração passa para "IF ESC"  facilitando a compreensão do código.

Cada nome de tecla possui uma constante com o prefixo CODE para ser usado na passagem de parâmetros, por exemplo criando um push button que deve retornar o código da tecla esc quando pressionado.

EXEC COBOLware Object PUSH-BUTTON
           CAPTION "~Processar"
           LINE 23 COLUMN 03
           KEY CODE-F2 TAB-OFF
END-EXEC

ACCEPT tecla FROM ESCAPE KEY

IF   F2
      PERFORM PROCESSAR
END-IF

Encontra-se em CWKEYS.CPY. Para obter a definição utilize o comando [COPY CWKEYS] logo após a declaração do campo receptor do código da tecla.
