EOL

Converte formato de arquivos texto DOS <=> Unix

Sintaxe

EOL [ DOS | Unix ]
[ FILE arquivo ]
[ STATUS filestatus ]

Argumentos

DOS

O arquivo será convertido para o formato DOS (Terminador de registro X”0D0A”).

Unix

O arquivo será convertido para o formato Unix (Terminador de registros X”0A”).

arquivo

Nome do arquivo texto ser convertido (Label) lteral ou variável de até 255 caracteres.

file status

Informa o código de status COBOL resultante da operação.

Exemplo

Codificação COBOL

EXEC COBOLware EOL Unix
            FILE "teste.txt"
 END-EXEC
