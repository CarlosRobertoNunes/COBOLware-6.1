RUN

Executor de programas avulsos

Formatos:

1

RUN Programa

Modo console API ou Unix.

2

RUNW Programa

Modo text Window.

3

RUNG Programa

Modo gráfico.

Onde Programa é o nome do programa a ser executado (sem extensão).

Testar um programa sem precisar passar pelo Gerenciador de sistemas é um recurso interessante para facilitar o trabalho de testes no ambiente de desenvolvimento e para rodar processos batch.

Para evitar que se abra uma espécie de "porta dos fundos" que violaria a segurança do sistema no ambiente dos usuários, será exigido usuário e senha. Este incômodo pode ser contornado com a variável de ambiente CWTEST=/U:Usuário /S:Senha /NOFRAME sendo que NOFRAME indica que a moldura do Gerenciador de sistemas não deve ser traçada no modo texto.
